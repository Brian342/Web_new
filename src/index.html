Lists

you can also use HTML to create lists. to create an ordered list, use the following tags :

  1. First thing to do
  2. Second thing to do
  3. Third thing to do

To create an unorded list, use the following tags:

  • Apples
  • Pears
  • Oranges

You can also nest lists within one another, like so:

  1. Go to the supermarket and buy
    • Apples
    • Pears
    • Oranges
  2. Go to the post office and send letter to Alice
